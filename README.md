# VF2 Subgraph Isomorphism: Sample Data Analysis

VF2 subgraph isomorphism finds all occurrences of a small query graph inside a larger target graph by systematically matching vertices and edges while respecting structural constraints. The algorithm is used here to detect every induced subgraph of G that is isomorphic to H, with results deduplicated across symmetric mappings to surface only distinct matches.
